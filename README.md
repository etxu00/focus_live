# focus_live
Plataforma de salas de trabajo para streamers basada en la técnica Pomodoro personalizable. Permite crear sesiones de enfoque con acceso vía QR/enlace, layouts dinámicos para OBS, encuestas en vivo y resúmenes de resultados. La herramienta ideal para sincronizar la productividad de tu comunidad con intervalos de trabajo y descanso a tu medida.
